# AutoClash

AutoClash 是一款基于 Android 平台的自动化代理组切换工具。它可以根据你当前所处的网络环境（如连接的 WiFi 或者使用的网络运营商），自动调用 Mihomo (Clash Meta) 的 API 来切换相应的策略组节点。

## ✨ 核心特性

- **基于环境的智能切换**：
  - **WLAN (WiFi)**：根据当前连接的 WiFi SSID 名称自动切换策略组。
  - **运营商 (ISP)**：根据当前使用的网络运营商（如中国电信、中国联通、中国移动）自动切换策略组。
- **灵活的规则引擎**：支持多条自动化规则、设置规则优先级。
